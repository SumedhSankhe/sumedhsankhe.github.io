Featured Project

NULISA Analysis Software (NAS)

Lead Developer & Architect

A comprehensive web application providing an intuitive interface for analyzing and visualizing data generated by the NULISA platform. NAS facilitates analysis of complex proteomics datasets, enabling researchers to gain insights quickly and efficiently. Built with enterprise-grade architecture including telemetry tracking, async operations, and comprehensive data quality controls.

Quality Control: Sample QC analysis and validation
Statistics: Differential expression analysis with volcano plots
Pathway Analysis: GSEA results with enrichment scores and interactive data tables

Technology Stack

R Shiny Docker Kubernetes (AKS) Azure Container Registry CI/CD Pipelines SQL testthat Modular Architecture shiny.telemetry renv Cypress Testing Async Processing NGS/Sequencing

Architecture & Major Modules

  • Project Module: 3 submodules for comprehensive project management and data initialization
  • Preview Module: 6 submodules + 2 nested levels for target, sample, covariate, and QC data inspection
  • QC Module: 4 submodules + 3 nested levels for precision analysis, normalization, and detectability assessment
  • Stats Module: 3 high-complexity submodules for model selection, summary generation, and visualization
  • Heatmap Module: Interactive data exploration capabilities
  • Additional Modules: Exit Record management, View Details, and Report Download

Technical Achievements

  • Orchestrated complex data flows across 20+ interconnected server modules
  • Implemented nested submodule architecture with up to 3 levels of depth
  • Managed complete CI/CD pipeline with Kubernetes deployment manifests and automated testing gates
  • Configured testthat unit tests with renv environment parity for pre-deployment validation
  • Integrated telemetry system tracking user interactions across all modules
  • Advanced data cleaning pipeline handling covariates, samples, and quality metrics
  • Asynchronous processing for computationally intensive statistical analyses
  • Comprehensive testing framework (testthat + Cypress) ensuring reliability across deployment pipeline
  • Production deployment on Azure Kubernetes Service with Azure Container Registry
