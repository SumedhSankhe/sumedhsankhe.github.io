# Portfolio Website

Personal portfolio showcasing bioinformatics software engineering projects, technical skills, and professional experience.

**Live Site:** [sumedhsankhe.github.io](https://sumedhsankhe.github.io)

## Built With

- HTML5, CSS3, JavaScript (ES6+)
- Custom CSS with CSS Variables for theming
- Responsive design
- Vanilla JavaScript (no frameworks)

## Features

- Dual theme support (light/dark mode)
- Fully responsive across all devices
- Accessible navigation (WCAG compliant)
- Interactive project demonstrations with GIFs
- Dynamic skills visualization with rotating package logos
- Optimized performance and fast load times

## Project Structure

```
.
├── index.html          # Homepage with hero and expertise sections
├── about.html          # Professional experience and education
├── projects.html       # Detailed project showcases
├── skills.html         # Technical skills with visualizations
├── blog.html           # Blog section (coming soon)
├── contact.html        # Contact information
├── styles.css          # Main stylesheet with theme support
├── script.js           # Core JavaScript functionality
├── favicon.svg         # Site favicon
└── assets/
    ├── gif/           # Project demonstration GIFs
    └── svg/           # Package logos and icons
```

## Technical Highlights

- **CSS Architecture:** Modular design with CSS custom properties for consistent theming
- **JavaScript:** Event-driven architecture with debounced scroll handlers
- **Accessibility:** Semantic HTML, keyboard navigation, ARIA labels, skip links
- **Performance:** Optimized images, minimal dependencies, efficient animations
- **Theme System:** Persistent theme preference with localStorage and system preference detection

## Local Development

Clone and serve locally:

```bash
git clone https://github.com/SumedhSankhe/sumedhsankhe.github.io.git
cd sumedhsankhe.github.io

# Serve with Python
python -m http.server 8000

# Or with Node.js
npx serve
```

Visit `http://localhost:8000` in your browser.
